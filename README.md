## How to run it backend

1) Clone this repository
2) cd backend
3) Install dependencies using below command :
    run npm install
4) Create database and connection in phpmyadmin and name it `ex-rate`
5) Exporting the sql database names `ex-rate.sql`
6) run npm start


## How to run frontend

1) Clone this repository
2) cd backend
3) Install dependencies using below command :
    run npm install
